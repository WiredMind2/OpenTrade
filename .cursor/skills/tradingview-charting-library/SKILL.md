# TradingView Charting Library

This skill bundles TradingView’s official **Charting Library Context** export: bundled `.d.ts` typings and markdown documentation excerpts.

## When to use it

- Wiring or debugging `TradingView.widget`, datafeeds, broker APIs, or chart options.
- Questions about types, callbacks, or documented behavior for the Charting Library.

## Reference file

Read **`charting-library-context.txt`** in this same folder. It is the verbatim dump from:

https://www.tradingview.com/charting-library-docs/charting-library-context.txt

Prefer searching that file (or opening the section you need) rather than guessing API names. For the live site, use the official Charting Library docs in a browser.

## Refreshing the snapshot

To update the text file, re-download from the URL above and replace `charting-library-context.txt`.
